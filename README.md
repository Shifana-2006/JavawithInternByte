# Student Management System (Java Console Project)

## Overview
This project is a simple console-based Student Management System built using Java and Object-Oriented Programming (OOP) concepts. It allows users to perform CRUD (Create, Read, Update, Delete) operations on student data.

## Features
- Add new student details
- View all registered students
- Search a student by roll number
- Update existing student information
- Delete a student record

## Technologies Used
- Java
- OOP (Classes, Objects, Encapsulation)
- Java Collections Framework (ArrayList)
